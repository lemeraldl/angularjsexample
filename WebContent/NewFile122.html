Using an object = GoodController

What's the most patriotic color?
Color inside isolated scope: {{uiState.myModel|json}}

The ng-if directive created an isolated scope.

This box is not in an isolate scope.
Color outside isolated scope: {{uiState.myModel|json}}
Updates from outside will appear inside even after a color is chosen inside.
